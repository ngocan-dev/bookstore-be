# 📚 Bookstore BE

Backend REST API for an online bookstore, built with Spring Boot.

---

## 🚀 Overview

Bookstore BE is a backend service that handles core functionalities such as book management, user operations, and order processing for an online bookstore system.

It provides RESTful APIs to support frontend applications.

---

## ✨ Features

- CRUD operations for books
- User authentication & management
- Order processing
- RESTful API design
- Database integration
- Clean layered architecture (Controller → Service → Repository)

---

## 🛠 Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Maven
- MySQL / H2
- Docker
